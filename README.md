# Random-Forest-Project
This project uses a Random Forest Classification model to analyze LendingClub loan data. The goal is to predict whether a borrower will fully repay their loan based on financial and credit-related features. The dataset includes information such as credit policy, interest rate, income, debt-to-income ratio, and credit score.

The project involves data exploration, visualization, feature engineering (handling categorical variables using dummy encoding), and building machine learning models including Decision Tree and Random Forest. The performance of both models is evaluated using confusion matrix and classification report to compare their accuracy and effectiveness.
